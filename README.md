# E-Commerce Store

Простой интернет-магазин на Django для лабораторной работы по информационным системам.

## Технологии

- **Backend**: Django 5.0.2
- **Database**: SQLite
- **Frontend**: Bootstrap 4.4.1, Django Templates

## Основные функции

- Каталог товаров с изображениями
- Корзина покупок (сохраняется через cookies/сессии)
- Оформление заказов
- Регистрация и авторизация пользователей
- Панель администратора Django

## Установка и запуск

1. **Создайте виртуальное окружение:**
   ```bash
   python -m venv venv
   venv\Scripts\activate
   ```

2. **Установите зависимости:**
   ```bash
   pip install Django==5.0.2
   ```

3. **Примените миграции:**
   ```bash
   python manage.py migrate
   ```

4. **Создайте суперпользователя (для админки):**
   ```bash
   python manage.py createsuperuser
   ```

5. **Запустите сервер:**
   ```bash
   python manage.py runserver
   ```

6. **Откройте в браузере:**
   - Магазин: http://127.0.0.1:8000/
   - Админка: http://127.0.0.1:8000/admin/

## Структура проекта

```
ecommerce/
├── ecommerce/          # Конфигурация проекта
├── store/              # Основное приложение
│   ├── models.py       # Модели: Customer, Product, Order, OrderItem
│   ├── views.py        # Представления
│   └── urls.py         # URL-маршруты
├── static/             # CSS, изображения
└── db.sqlite3          # База данных
```

## URL маршруты

| Маршрут | Описание |
|---------|----------|
| `/` | Главная страница магазина |
| `/cart/` | Корзина |
| `/checkout/` | Оформление заказа |
| `/signup/` | Регистрация |
| `/login/` | Вход |
| `/logout/` | Выход |
| `/admin/` | Панель администратора |
